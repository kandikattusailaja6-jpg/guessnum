### Number Guessing Game – Theory (15 Points)

1. The Number Guessing Game is a simple Python-based interactive game.
2. It uses the `random` module to generate a random number.
3. The secret number is selected between **1 and 200**.
4. The player is informed about the range of the secret number.
5. The program asks the user to enter a guess.
6. User input is accepted using the `input()` function.
7. The entered value is converted into an integer using `int()`.
8. A `while` loop is used to continue the game until the correct guess is made.
9. The program compares the guessed number with the secret number.
10. If the guess is less than the secret number, a **"Too low"** message is displayed.
11. If the guess is greater than the secret number, a **"Too high"** message is displayed.
12. These hints help the player make a better guess in the next attempt.
13. When the guessed number matches the secret number, the player wins the game.
14. A congratulatory message is displayed to the user.
15. The program then terminates, demonstrating the use of random numbers, loops, conditions, and user input in Python.
